¿Qué es CSS?

En una página web se escribe código HTML, con el que se que crea el contenido de una web. Por otro lado, existe el código CSS, que unido al código HTML permite darle forma, color, posición (y otras características visuales) a un documento web.

Las siglas CSS (Cascading Style Sheets) significan «Hojas de estilo en cascada». Se le denomina estilos en cascada porque se lee, procesa y aplica el código desde arriba hacia abajo y parten de un concepto simple pero muy potente: aplicar estilos (colores, formas, márgenes, etc...) a uno o varios documentos (generalmente documentos HTML, páginas webs) de forma automática y masiva.

¿Por qué usar CSS?

La idea de CSS es la de utilizar el concepto de separación de presentación y contenido. Este concepto se basa en que, como programadores, lo ideal es separar claramente el código que escribimos. ¿Por qué? Porque con el tiempo, esto hará que el código sea más fácil de modificar y mantener.

La idea es la siguiente:
  • Los documentos HTML (contenido) incluirán sólo información y datos, todo lo relativo a la información a transmitir.
  • Los documentos CSS (presentación) inclurán sólo los aspectos relacionados con el estilo (diseño, colores, formas, etc).

¿Cómo enlazar un documento CSS a un HTML?

Para enlazar un documento CSS solo necesitas poner <link rel="stylesheet" href="nombre_archivo_css"> en la sección head del archivo HTML

Selectores

En CSS los selectores se utilizan para delimitar los elementos HTML de nuestra página web a los que queremos aplicar estilo. Hay una amplia variedad de selectores CSS, lo que permite una gran precisión a la hora de seleccionar elementos a los que aplicar estilo.

Un selector es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS. El elemento o los elementos seleccionados por el selector se denominan sujeto del selector.

Tipos de selectores

Selector universal

Es útil cuando deseas aplicar un estilo a todos los elementos en una página web sin excepción.

  • Sintaxis: *{atributo:valor;}
  • Ejemplo: <* { color: grey; }">

Selector de tipo

Selecciona todos los elementos que coincidan con el nombre de la etiqueta HTML.

  • Sintaxis: etiqueta { atributo:valor }
  • Ejemplo: <p {color: green;}">

Selector de clase

Selecciona todos los elementos que coincidan con las etiquetas HTML que contengan el atributo class

  • Sintaxis: .clase { atributo:valor }
  • Ejemplo: <.blend{color: red;}">

Selector de id

El selector identificador utiliza el atributo id para seleccionar un elemento. Solo puede haber un elemento con un id dado en un documento.

  • Sintaxis:#id { atributo:valor }
  • Ejemplo: #cent {color: blue;}">

Propiedades de CSS

Las propiedades son los valores que se les pueden asignar a los elementos de la página para modificar su apariencia. Algunas propiedades son:

Descripción

Valores de CSS

Los valores son las especificaciones que se asignan a las propiedades para definir el estilo de un elemento. Todas las propiedades necesitan un valor para funcionar correctamente. Algunos ejemplos son:

Valores de color

Pueden definirse con nombres, códigos hexadecimales, RGB o HSL:

  • color: blue;
  • color: #ff0000;
  • color: rgb(255, 0, 0);
  • color: hsl(0, 100%, 50%);
Valores numéricos

Se utilizan en medidas como width, height, margin, etc:

  • width: 200px;
  • height: 50%;
  • font-size: 12px;
Valores de texto

Se utilizan en propiedades relacionadas con el texto como font-family, font-weight, etc:

  • font-family: "Arial";
  • font-weight: bold
  • font-style: itallic;
Valores de tamaño

Se utilizan en propiedades relacionadas a tamaños como font-size, margin, border, etc:

  • font-size: 2em;
  • margin: 10;
  • border: 50px;
