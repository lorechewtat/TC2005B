Comandos de terminal

¿Qué es la terminal?

La terminal o línea de comandos, es una interfaz de usuario en la que se pueden introducir comandos de texto para interactuar con el sistema operativo de un ordenador. En lugar de utilizar una interfaz gráfica de usuario (GUI) con ventanas y botones, la terminal de comandos permite al usuario interactuar con el sistema operativo utilizando comandos de texto simples.

Comandos utiles para usar en la terminal:

Comando

Descripción
clear Sirve para limpiar la consola.
cd "nombre del directorio" Sirve para cambiar de directorio. Si pones cd.. te regresa al directorio anterior.
ls Sirve para ver los contenidos del directorio en donde estás actualmente. Si pones ls -al, te mostrará los contenidos en forma de lista.
pwd Sirve para imprimirte en que directorio estas ahorita.
clear Sirve para limpiar la consola.
mkdir "nombre del directorio" Sirve para crear un nuevo directorio. Se recomienda no poner espacios entre palabras, sino _.
touch "nombre del archivo" Sirve crear un nuevo archivo.
  • Con la letra i, entras al modo insert.
  • Con la letra esc, sales del modo insert.
  • Con :wq, guarda los cambios del archivo y vuelve a la terminal.
  • Con :w, guarda los cambios del archivo.
  • Con :q!, regresa a la terminal sin guardar los cambios.
vim "nombre del archivo" Sirve para abrir el editor de texto dentro de la terminal.
cat "nombre del archivo" Sirve para mostrar el contenido de un archivo.
code "nombre del archivo" Sirve para abrir el archivo en Visual Studio Code.
cp "nombre del archivo" "nombre del directorio" Sirve copiar un archivo a otro directorio.
rm "nombre del archivo" Sirve para borrar un archivo.
rm -r "nombre del directorio" Sirve para borrar un directorio. Si pones rm -rf, forzas que se borre.
mv "nombre del archivo" "directorio" Sirve para mover un archivo a un directorio en especifico.
También en algunas terminales si pones man, te mostrará el manual de usuario.
